# v0.1.4 Release Notes

**Release Date**: 2025-12-09

## Summary

This release adds jq, yq-go, and git to sbom-tools for CI workflow support, and improves the manual signing workflow DX.

## Changes

### sbom-tools CI Workflow Support

Added essential CLI tools to sbom-tools for complete CI workflow support:

| Tool  | Version | Purpose                          |
| ----- | ------- | -------------------------------- |
| jq    | 1.7.1   | JSON processing (SBOM output)    |
| yq-go | 4.44.5  | YAML processing (CycloneDX YAML) |
| git   | 2.47.3  | Clone repos for scanning         |

This supports two primary use cases:

1. **CI Container**: Using sbom-tools as the base image for CI jobs
2. **Local Tool Substitute**: Running the container instead of installing tools locally

### DX Improvements

- `make release-digests` now auto-fetches and displays image digests with ready-to-copy cosign commands
- RELEASE_CHECKLIST.md updated with environment variables:
  - `PGP_KEY_ID` for GPG signing
  - `MINISIGN_KEY` for minisign key path
  - `GONEAT_DIGEST` / `SBOM_DIGEST` for image digests
- Backslash line continuations for easier copy-paste in signing commands

## Verification

### Cosign (keyless)

```bash
cosign verify \
  --certificate-oidc-issuer https://accounts.google.com \
  --certificate-identity-regexp ".*@.*" \
  ghcr.io/fulmenhq/sbom-tools:v0.1.4
```

### GPG

```bash
curl -LO https://github.com/fulmenhq/fulmen-toolbox/releases/download/v0.1.4/SHA256SUMS-sbom-tools
curl -LO https://github.com/fulmenhq/fulmen-toolbox/releases/download/v0.1.4/SHA256SUMS-sbom-tools.asc
curl -LO https://github.com/fulmenhq/fulmen-toolbox/releases/download/v0.1.4/fulmen-toolbox-release-signing-key.asc

GPG_TMPDIR=$(mktemp -d)
gpg --homedir "$GPG_TMPDIR" --import fulmen-toolbox-release-signing-key.asc
gpg --homedir "$GPG_TMPDIR" --verify SHA256SUMS-sbom-tools.asc SHA256SUMS-sbom-tools
rm -rf "$GPG_TMPDIR"
```

### Minisign

